# AI 生成闪卡使用指南

> 文档同步日期：2026-07-16
> 变更说明：Plan B 修复后行为——加强接地、预览校验、尽力回滚与残留 ID 提示。

## 功能概述

**AI 生成闪卡** 根据**当前光标块**文本，调用 OpenAI 兼容服务生成 Basic/Cloze 草稿；在弹窗中检查、编辑、勾选后确认写入。

实现见 [SRS_AI模块.md](./SRS_AI模块.md)。

## 配置

| 项 | 说明 |
| --- | --- |
| API Key | 必填 |
| API URL | 完整 chat/completions URL（勿用 Ollama `/api/chat`） |
| AI Model | 模型名 |

命令面板 **「SRS: 测试 AI 连接」** 可探活（有超时）。

## 使用

1. 光标放在有文本的块上
2. `/AI 生成闪卡` 或命令面板「SRS: AI 生成闪卡」
3. 选 Basic/Cloze 与最多 1/3/5 张 → 生成草稿（可取消；错误保留弹窗）
4. 预览中编辑/勾选；已选无效草稿会显示行内错误，保存按钮不可用
5. **保存 N 张** 才写库；关闭不写

若对话框已打开，再次触发命令会提示先关闭，避免覆盖进行中的生成或预览。

## 写入与失败

- 成功：整批一步撤销
- 失败：尝试删除本批新卡（尽力而为，非硬事务）；若仍有残留，弹窗/通知会列出块 ID，请手动检查删除

## 注意事项

1. 仅当前块文本
2. 模型输出须接地（依据摘录、Basic 答案来自依据、Cloze 全文为源摘录）
3. 语言跟随源文本
4. Azure 仅 `api-key` 头时不支持

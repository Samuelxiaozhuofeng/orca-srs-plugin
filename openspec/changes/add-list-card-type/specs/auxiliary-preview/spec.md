## ADDED Requirements

### Requirement: 辅助预览不计入统计
系统 SHALL 将“辅助预览”视为非正式复习：允许评分，但不更新 SRS 状态、不写入复习日志、不计入任何正式统计。

#### Scenario: 辅助预览评分不产生收益
- **GIVEN** 用户在列表卡复习中触发了“辅助预览”阶段
- **WHEN** 用户对辅助预览条目进行评分
- **THEN** 系统不应更新该条目的 SRS 状态
- **AND** 系统不应写入复习日志记录
- **AND** 系统不应将该评分计入正式统计

### Requirement: 辅助预览提示明确
系统 SHALL 在 UI 中明确提示用户当前处于“辅助预览”阶段，以及“评分不计入统计”的规则。

#### Scenario: UI 提示
- **GIVEN** 系统正在展示辅助预览条目
- **WHEN** 用户查看复习界面
- **THEN** 系统应展示“辅助预览，不计入统计”等明确提示

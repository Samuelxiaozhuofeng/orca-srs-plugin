orca-srs 插件说明页

本仓库是 Orca Notes 插件，不能作为独立 SPA 在浏览器中运行。 本页为静态说明桩，不加载 src/main.ts。

推荐开发与安装流程：

  1. Node.js 20.19+
  2. npm ci
  3. npm run build（仅仓库内 dist/）
  4. ORCA_PLUGIN_ROOT=/absolute/path/to/orca/plugins/orca-srs npm run deploy:local
  5. 在 Orca 中启用插件

发布布局：npm run release:stage → release:verify →（版本非 0.0.0 时）release:zip。

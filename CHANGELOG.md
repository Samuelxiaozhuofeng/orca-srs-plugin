# 更新日志

## [Unreleased]

## [1.0.4] - 2026-07-22

### 新增
- AI 快捷交互与 Prompt 管理：工具栏快捷交互、Prompt 管理器、Markdown 解析与插入计划
- AI 后台任务与块内预览：加载指示、预览树操作、「保留此块」、离开面板自动 dismiss 预览
- AI / Firecrawl 服务设置面板：原生联网搜索与 reasoning effort 等选项
- 块解释快捷键与 UI 增强（渐进阅读会话）

### 变更
- EPUB / 网页导入从工具栏按钮迁移为 slash 命令入口
- AI 插入文本净化：正确处理 Markdown 链接与数字脚注，避免误链库内低 id 块

### 已知限制
- EPUB ZIP 中央目录声明体积尚未独立预检，仍依赖实际解压预算限制。
- EPUB、网页导入和 AI 保存的写入阶段尚未统一为可中断事务。

## [1.0.2] - 2026-07-20

### 新增
- 渐进阅读块解释（Block Explain）：会话内对当前块生成解释、侧栏内容与追问，并写入规范化子块
- 块解释 UI 与主题变量：`IRBlockExplainInline` / controller，配套 `ir-workspace` 样式

### 变更
- 基础卡答案编辑改为单一 live block，避免双块编辑错位
- 复习内嵌块默认 `initiallyCollapsed: false`，折叠笔记中仍可见正文
- AI 制卡 grounding 校验增强 Markdown 链接识别，并更新 service prompts

### 已知限制
- EPUB ZIP 中央目录声明体积尚未独立预检，仍依赖实际解压预算限制。
- EPUB、网页导入和 AI 保存的写入阶段尚未统一为可中断事务。

## [1.0.1] - 2026-07-20

### 新增
- 闪卡主页单页布局：`HomeSummaryBar` 可点击统计导航、`CardFrame` 状态色条与卡组列表体验增强
- 渐进阅读重要性：用户可见「重要性」文案，建书/导入三档绝对档位与会话相对微调（`Alt+P`）
- 渐进阅读摘录近上下文与章节浏览：定位高亮、上下文 hide-self、browse 返回 focus
- 渐进阅读混合摘录处理：二次挖空（`keep_extract`）并保留 IR 调度
- 顺序阅读完成与归档：章节完成后资料库保留结构节点、正文不删
- 会话动作栏与更多面板：主栏「下一篇 / 摘录|挖空 / 重要性 / 完成 / ⋯」，更多面板 dismiss 逻辑

### 变更
- 闪卡主页移除 Dashboard / 全页统计 Tab，统一为 `FlashHomePage` 单页
- 渐进阅读会话创建路径只读装配队列；资料库「一键溢出推后」与会话启动解耦
- 修复隐藏 IR 工作区布局（hideable 与 inline flex 冲突）

### 已知限制
- EPUB ZIP 中央目录声明体积尚未独立预检，仍依赖实际解压预算限制。
- EPUB、网页导入和 AI 保存的写入阶段尚未统一为可中断事务。
